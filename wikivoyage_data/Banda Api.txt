
Banda Api is an island in Banda Islands. 


Understand[edit]
Gunung Api Banda and Belang (traditional Banda boat)
Banda Api is a volcanic island or high island and it is known as Gunung Api Banda.

Get in[edit]
Rent a small boat from Banda Neira
